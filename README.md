# AWS-CI-CD-Pipeline-Automated-Website-Deployment
This project demonstrates a complete CI/CD pipeline that automatically deploys a static website from GitHub to AWS S3 using AWS CodePipeline.
Architecture

GitHub → AWS CodePipeline → Amazon S3 → Live Website

⚙️ Services Used
AWS CodePipeline (CI/CD automation)
Amazon S3 (Static website hosting)
IAM (Access control)
GitHub (Source code management)
🔄 Workflow
Code pushed to GitHub
CodePipeline detects changes
Pipeline deploys files to S3
Website updates automatically
